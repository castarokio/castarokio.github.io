# Castarokio Portfolio

Static portfolio site prepared for GitHub Pages.

## Files

- `index.html` main page
- `styles.css` site styling
- `script.js` mobile menu and footer year
- `assets/profile.jpg` profile image

## Publish to GitHub Pages

1. Create a new GitHub repository.
2. Upload these files or push the folder with Git.
3. In GitHub, open `Settings` -> `Pages`.
4. Set source to `Deploy from a branch`.
5. Choose branch `main` and folder `/ (root)`.
6. Save and wait for GitHub Pages to publish.

## Customize

- Update contact links in `index.html`
- Replace project descriptions with your real work
- Swap `assets/profile.jpg` with your own image
